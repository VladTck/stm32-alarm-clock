# STM32 Alarm Clock

This project is a simple embedded alarm clock developed for an STM32 microcontroller.  
It was created as part of a university microcontroller programming course.

The system works as a digital clock that counts hours, minutes and seconds, displays the time on an LCD screen, and allows the user to set an alarm.

---

## Features

- Real-time counting of hours, minutes and seconds
- Alarm function with sound signal
- LCD display output
- Setting current time and alarm time
- Button control using hardware interrupts
- Timer-based time counting

---

## Hardware

Example hardware configuration used in the project:

- STM32 microcontroller
- LCD 16x2 display
- buttons for control
- buzzer (for alarm signal)

---

## Technologies

- C programming language
- STM32 microcontroller
- STM32CubeIDE
- Embedded systems programming

---

## Controls / Functionality

The system allows the user to:

- set the current time
- set an alarm time
- trigger an alarm with sound signal
- display time on LCD

Buttons are handled using interrupts to ensure responsive input handling.

---

## Project Purpose

The goal of this project was to practice embedded systems programming, including working with timers, interrupts, and microcontroller peripherals.
